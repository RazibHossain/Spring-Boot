1. Stop everything
docker-compose down
2. Remove old containers + network + orphan services (IMPORTANT for clean start)
docker-compose down --remove-orphans
3. Remove old volumes (THIS resets MySQL data)
docker-compose down -v

👉 Use this only if you want full reset (recommended for dev)

4. Clean unused Docker cache (optional but good)
docker system prune -f
5. Pull latest images
docker-compose pull
6. Start everything fresh
docker-compose up -d
7. Watch logs (very important)
docker-compose logs -f

or for single service:

docker-compose logs -f order-service
